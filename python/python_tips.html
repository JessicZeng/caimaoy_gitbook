      "plugins": [
          "disqus"
      ],
      "pluginsConfig": {
          "disqus": {
              "shortName": "caimaoygitbook"
          }
      }
  }

      ❯ python -m json.tool README.md

  • 简易Web Server 通过 Python 快速共享文件,进入到你要共享文件的目录下并在命令行中运行下面的代码：

      # Python 2
  python -m SimpleHTTPServer

  # Python 3
  python -m http.server
