# gitbook

- 安装 node.js
- 安装 gitbook
    ```
    npm install gitbook-cli -g
    ```
    安装之后
    ```
    ❯ gitbook -V
    2.1.2
    ```
- 初始化 gitbook
    ```
    gitbook init
    ```

- 启动 gitbook server
    ```
    gitbook serve
    ```

- [gitbook 简明教程](http://www.chengweiyang.cn/gitbook/index.html)
